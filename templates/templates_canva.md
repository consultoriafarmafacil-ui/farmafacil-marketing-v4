## Como Acessar no Canva

1. Acesse [canva.com/starred](https://canva.com/starred) para ver os favoritos
2. Pesquise pelo código do template (ex: `DAHEmyA0`)
3. Ou acesse: `canva.com/design/[CÓDIGO]/edit`

> ⚠️ **Atenção:** As imagens do Canva precisam ser baixadas manualmente a cada sessão  
> pois os servidores externos não têm acesso direto às URLs `media-public.canva.com`

## Identidade Visual — Referência Rápida

```css
--fundo:    #F4F7F9;   /* cinza suave — background dos posts */
--petroleo: #0D2B3E;   /* azul petróleo escuro — texto principal */
--mint:     #00C9A7;   /* verde mint — destaques e CTAs */
--fonte:    "Tinos", serif;   /* Bold Itálico */
```

### Paleta Complementar (uso secundário)
- Branco `#FFFFFF` — fundos alternativos
- Cinza médio `#6B7280` — textos secundários
- Verde farmácia `#16A34A` — ícones de saúde
